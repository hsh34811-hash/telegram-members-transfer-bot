# 🤝 دليل المساهمة

## 🚀 خطوات المساهمة

**1. Fork المشروع**

**2. أنشئ فرع جديد**
```bash
git checkout -b feature/اسم-الميزة
```

**3. ارفع التغييرات**
```bash
git commit -m "إضافة: وصف التغيير"
git push origin feature/اسم-الميزة
```

**4. افتح Pull Request**

## 📋 قواعد
- لا ترفع ملف `jello.ini` أو ملفات `.session`
- اختبر التغييرات قبل الرفع
